# 2026.6.22-6.24 记录三天两晚的承德旅行
时间在毕业典礼和端午节之后，在室友的热情邀请下和同班同学一起去承德游玩，很感谢室友以及室友父母、叔叔婶婶的热情招待!

# 6.22
从北京朝阳站乘坐G3509前往承德南站，中午11.20抵达，承德的天气很不错，那个时候的扬州连着下了几天的雨，承德是晴天。

<div style="width: 30%; margin: 0 auto;">

![抵达承德](./photos/抵达承德.jpg)
</div>

第一顿饭是一百家子拨御面，点了四碗面条和当地的特色菜————血肠和碗坨。面条的汤挺好喝的，像牛肉清汤的味道，血肠和碗坨是荞麦面做成的，蘸蒜泥后味道很不错。
<div style="width: 60%; margin: 0 auto;">

![一百家子拨御面](./photos/第一天午饭.jpg)
</div>

这家店离避暑山庄很近，隔着一条河，我们步行前往避暑山庄门口，中午的太阳还是有一点晒的。
<div style="width: 60%; margin: 0 auto;">

![前往避暑山庄的路上](./photos/前往避暑山庄的路上.jpg)
</div>
避暑山庄斜对面是杏好遇见你，承德的特产是杏仁。我们尝了杏仁饼、杏仁酪和杏仁豆腐，记忆里承德带着杏仁的味道。
<div style="width: 60%; margin: 0 auto;">

![杏好遇见你](./photos/杏好遇见你.jpg)
</div>
随后我们抵达了避暑山庄丽正门入口，刚进门是宫殿区。避暑山庄的建设布局和中国的地理布局对应，西边是山峦区，东北方向是平原区，东南方向是湖泊区，分别对应华北平原和江南水乡。
经过宫殿区后，我们乘车上山，在第二站下车，有小鹿。
<div style="width: 60%; margin: 0 auto;">

![小鹿](./photos/小鹿.jpg)
</div>
山里很凉快，有风吹过，走着走着就到了城墙，可以远眺小布达拉宫和班禅行宫。
<div style="display: flex; gap: 10px; justify-content: center;">

![小布达拉宫（远）](./photos/小布达拉宫1.jpg)
![小布达拉宫（近）](./photos/小布达拉宫2.jpg)
</div>
这么近那么美，周末到河北。
<div style="width: 60%; margin: 0 auto;">

![](./photos/小布达拉宫3.jpg)
![](./photos/小布达拉宫4.jpg)
</div>
沿着路走到下一上车点，坐车下山。接下来游览的是湖泊区与平原区。
<div style="width: 60%; margin: 0 auto;">

![小桥流水](./photos/湖泊区.jpg)
![还珠格格漱芳斋](./photos/烟雨楼.jpg)
![平原区](./photos/平原区.jpg)
</div>
快到下午四五点下起了雨，承德的雨来的快去的也快，这三天都是这样，下起雨躲一会，雨就停了。
<div style="width: 60%; margin: 0 auto;">

![荷叶](./photos/荷叶.jpg)
![承德的雨](./photos/承德的雨.jpg)
</div>
晚上ch的爸爸妈妈还有叔叔婶婶招待了我们，在元宝山，饭菜很好吃，吃得也很饱，感谢招待！
吃完后步行去路对面的康熙大典演出，实景演出，很震撼。
<div style="width: 60%; margin: 0 auto;">

![](./photos/康熙大典1.jpg)
![康熙大典](./photos/康熙大典2.jpg)
</div>

# 6.23
第二天的安排是爬金山岭长城。
早饭吃了牛肉丸子汤、杏仁茶、碗坨和烧卖，补充好能量后坐大巴前往金山岭长城。
<div style="width: 60%; margin: 0 auto;">

![](./photos/第二天早饭.jpg)
![金山岭长城入口](./photos/金山岭长城入口.jpg)
</div>

这是我第一次爬长城，在天津四年也没去爬过北京的八达岭长城。爬的时候身体累累的，但看到的风景很美。
<div style="width: 60%; margin: 0 auto;">

![](./photos/长城3.jpg) 
![](./photos/长城2.jpg)
![](./photos/长城6.jpg)
</div>
爬到最后一段是古长城，还没有修葺，两边没有护栏，也很陡峭，我要手脚并用，重心放在地上才敢爬上去。
<div style="width: 60%; margin: 0 auto;">

![](./photos/长城7.jpg) 
![](./photos/长城8.jpg)
![](./photos/长城9.jpg)
![](./photos/长城10.jpg)
</div>
爬上去后也不敢站直腿，眼前的是不断的山，南方的孩子没有见过绵延不断的群山。
<div style="width: 60%; margin: 0 auto;">

![](./photos/长城11.jpg)
![](./photos/长城12.jpg)
![](./photos/长城15.jpg)
</div>
长城上还遇到两只小猫，很亲人。
<div style="width: 60%; margin: 0 auto;">

![](./photos/长城13.jpg)
![](./photos/长城14.jpg)
</div>

之后从后川楼下山，返程回市区大概四点左右。ch带着我们去承德的特产店逛了一下，给女朋友带一点这边的特产。

晚饭ch请我们吃了烧烤，点了很多，每次ch请客都会点好多饭菜，吃得特别饱。

<div style="width: 60%; margin: 0 auto;">

![](./photos/第二天晚饭.jpg)
</div>
可能是这几天吃得很饱，晚上回去上厕所把马桶拉堵了...
冲了好几次水也冲不下去，最后用马桶搋子疏通了，又增加生活经验了。晚上打了一会麻将，没想到最后居然是lbx赢得最多。
<div style="width: 60%; margin: 0 auto;">

![](./photos/麻将.jpg)
</div>

# 写在最后
在北京10号线草桥站分别，意味着我的本科真的结束了，不会再在天津学习生活，也和大学同学告别了。分别总是伤感的，我觉得大学毕业比高中毕业更让我难以接受，高中毕业的我想着去远方的世界看看，这也意味着，我与远方的朋友产生羁绊后，总要面对离别，于是和大学的好朋友总是流着泪说再见。十几年的学习生活，我似乎总在说再见，从初中到高中，再从高中到大学，再从大学到毕业，真正能再见到的朋友，也只有一只手能数得过来，每个人只能陪我走过人生中的一段路。但我想，我是幸运的，我很幸运能够遇见自己的每一位朋友，我很感激命运将你们带到我的面前，我也很感激你们，愿意成为我的朋友，愿意和我一起玩，愿意对我那么好。我想，我们一定会再见面的，一定会的。
> 祝愿大家都前程似锦。
